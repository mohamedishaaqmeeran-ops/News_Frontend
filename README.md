# News_Frontend

## Created a real-time news alert system that delivers breaking news updates and important notifications to users. Implement features for customizing alert preferences, such as news categories and frequency. Include tools for sending notifications via email, or push notifications.

## If news is created by journalist, user will receive notification based on his preference. Once a new user is registered, they will receive welcome mail.

#### Tech stack used: React and Tailwind css

##### to run : npm install....npm run dev
